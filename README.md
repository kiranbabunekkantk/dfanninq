# cupoc

**CU**DA-based **Po**int **C**loud library

## Core Features

* Point cloud calculations using CUDA
* [Open3D](https://github.com/intel-isl/Open3D)-like API

## Installation

```
pip install cupoc
```

```
git clone https://github.com/neka-nat/cupoc.git --recurse
cd cupoc
mkdir build
cd build
cmake ..; make -j
```
